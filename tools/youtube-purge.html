Get Your Subscriptions File
Google Takeout · ~2 minutes
You need to download your YouTube subscription list. Follow these steps exactly:
  1. Go to takeout.google.com and sign in to your YouTube account
  2. Click Deselect all at the top
  3. Scroll down, find YouTube and YouTube Music and check it
  4. Click the All YouTube data included button -> click Deselect all -> check only subscriptions -> click OK
  5. Scroll down -> click Next step -> Create export
  6. Download the zip -> unzip it -> find subscriptions.csv inside the YouTube folder
⚠️ It must be the file named exactly subscriptions.csv - not any other YouTube file.
Upload subscriptions.csv
Last step!
Upload the subscriptions.csv file you just downloaded. That's all you need to get started.

Unsubscribe Script — channels
This script will only unsubscribe channels you marked Remove. Everything else stays untouched.
How to use:
1. Go to youtube.com/feed/channels
2. Press F12 (Windows) or Cmd+Option+J (Mac)
3. Click the Console tab, type allow pasting, press Enter
4. Paste the script below and press Enter
5. Leave the tab open until you see Done!
